# Chapter 12: Endeavors

## Setting the Scene

At the start of an endeavor, the GM:
1. Defines the **objective** and **circumstances**
2. Sets an optional **collective threshold** (usually kept secret)
3. Outlines any **special rules** in effect

---

## Order of Endeavors

Like conversations, endeavors unfold in **flexible rounds**. Each player describes what their character does; the GM resolves it, usually with a test. One "round" ends when all players have had a turn.

---

## Collective Thresholds

Tracks total **successes** and **failures** to determine when the endeavor concludes.

If you reach enough successes: goal attained.  
If you accumulate too many failures first: endeavor fails.

### Setting the Threshold

| Complexity | Example | Successes | Failures |
|------------|---------|-----------|---------|
| Simple | Apprehend a criminal (Pursuit) | 4 | 3 |
| Basic | Determine function of a mysterious fabrial (Discovery) | * | 4 |
| Average | Activate an Oathgate in occupied territory (Mission) | 8 | 4 |
| Intricate | Locate a lost ancient city (Exploration) | 9 | 4 |
| Insurmountable | Evacuate thousands before enemy troops arrive | 11 | 5 |

*Discovery uses an investigative web instead of a success threshold.

> **Failing Forward:** Failures should never stop the story — they open different plotlines. Always plan both outcomes.

---

## Discovery Endeavors

*Characters investigate mysteries. Progress is tracked through an **investigative web**, not a flat success count.*

### Investigative Web
A map of clues and leads:
- **Leads** — starting points (2–4 revealed at opening)
- **Lead lines** — each lead connects to a series of clues
- Each clue requires a relevant skill test to uncover
- Multiple paths to the solution
- Failed tests use up failure threshold slots

If you run out of failures, you can attempt a **hunch** — guess the answer. Correct = success. Wrong = the story takes a twist.

**Discovery Skill Examples:**

| Skill | Use |
|-------|-----|
| Deduction | Interpret an existing clue |
| Insight / Perception | Find a clue from physical evidence |
| Intimidation / Persuasion | Extract information from someone |
| Lore | Recall relevant historical facts |
| Stealth / Survival | Shadow or track an elusive suspect |

---

## Exploration Endeavors

*Traverse unpredictable or hazardous environments.*

### Running an Exploration
The GM describes the setting in sensory detail. Players face environmental hazards, resource management, and navigation challenges.

**Exploration Skill Examples:**

| Skill | Use |
|-------|-----|
| Agility / Stealth | Avoid dangerous flora or fauna |
| Athletics | Climb to a vantage point or push through terrain |
| Crafting | Build a fire, set a trap, fashion a crossing |
| Deduction / Perception | Watch for threats, identify water sources |
| Discipline | Endure low rations or dehydration |
| Leadership | Bolster group confidence during difficult travel |
| Lore | Apply biology or geology knowledge |
| Medicine | Identify tainted food or infected water |
| Survival | Animal handling, tracking, hunting |

### Special Rule: Against the Odds

Once per Exploration, the GM may introduce a **major peril**. Every character makes a relevant test. Failing characters suffer the peril's effects:

| Peril | DC | Failure Effect |
|-------|-----|---------------|
| Dehydration | 14 | Lose all but 1 focus |
| Drained Spheres | 10 | Half your spheres go dun |
| Lost Energy | 14 | Exhausted [−1] |
| Obscured Senses | 18 | Disoriented for remainder of scene |
| Ruined Clothing | 15 | Disadvantage on Persuasion/Leadership; armor loses Presentable |
| Tainted Food | 12 | Take 2d4 vital damage |

## Mission Endeavors

*Urgent, tactically complex scenes — infiltrations, heists, rescues.*

### Running a Mission
The GM:
1. Defines primary and secondary objectives
2. Describes the location and obstacles
3. Sets a target timeline (see Ticking Clock)
4. Prepares robust Complication outcomes

Raise the stakes on most rolls to maintain urgency.

**Mission Skill Examples:**

| Skill | Use |
|-------|-----|
| Agility | Navigate unconventional paths (ledges, rafters) |
| Athletics | Quietly incapacitate guards |
| Crafting | Deactivate a fabrial alarm, improvise tools |
| Deception | Convince a sentry you belong there |
| Insight | Determine when guards have their guard down |
| Leadership | Coordinate the team |
| Lore | Find a path through architectural knowledge |
| Stealth | Move silently, avoid detection |
| Thievery | Lockpick, pickpocket a key, deactivate traps |

### Special Rule: The Ticking Clock
The GM sets a **target timeline**. Complications can advance the clock. Once the timeline is exceeded:
- Issues of increasing severity compound each round
- PCs may choose to cut their losses and escape

---

## Pursuit Endeavors

*Chase scenes — on land, air, or sea. One party pursues; the other flees.*

### Special Rule: Closing the Distance

Instead of a standard success threshold, the GM tracks a **distance track**:

| Point | Description |
|-------|-------------|
| **Starting Distance** | Initial gap between parties |
| **Escape Distance** | How far the target needs to be to escape |
| **Current Distance** | Adjusted each round based on test outcomes |

- Pursuer succeeds: gap closes by 1
- Pursuer fails: gap widens by 1
- Target succeeds: gap widens by 1
- Target fails: gap closes by 1

**Distance Examples:**

| Scenario | Start | Escape |
|----------|-------|--------|
| Running through a keep's hallways | 2 | 4 |
| Fleeing a chasmfiend in the chasms | 3 | 6 |
| Chasing an agent across Rall Elorim | 4 | 8 |

**Pursuit Skill Examples:**

| Skill | Use |
|-------|-----|
| Agility | Weave through crowds (can use opposed test) |
| Athletics | Push past obstacles (can use opposed) |
| Deception | Create a false trail (can use opposed) |
| Deduction / Insight | Identify shortcuts; guess next move |
| Intimidation / Persuasion | Convince bystanders to help |
| Perception | Spot a hidden pursuer; oppose Stealth |
| Stealth | Duck out of sight (can use opposed) |
| Thievery | Break into a building to gain/lose distance |
